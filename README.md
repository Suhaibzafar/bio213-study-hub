# BIO 213 — Final Exam Study Hub (Chapters 7–9)

An interactive study site for the BIO 213 final exam, covering **Chapter 7 (DNA Structure & Gene Function)**, **Chapter 8 (DNA Replication, Binary Fission & Mitosis)**, and **Chapter 9 (Sexual Reproduction & Meiosis)**.

## Features
- **Comprehensive notes** for every section of Ch. 7–9.
- **Chapter quizzes** — a randomized multiple-choice quiz for each chapter, with instant feedback and explanations.
- **Cumulative mock final** — 40 random questions across all three chapters.
- **Flashcards** — flip-card drills on key terms, filterable by chapter.

## Use it
Just open the live site (GitHub Pages) — no install needed. To run it locally instead:

```bash
python -m http.server 8213
```

Then visit `http://127.0.0.1:8213/`. The site is plain HTML/CSS/JS and also works by opening `index.html` directly.

Content based on *Hoefnagels, Biology: Concepts & Investigations*.
